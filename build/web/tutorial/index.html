Tutorial: JS Framework for enhanced Thumbnail

Click on the image as it will open a new link to the ranger's bio

To learn more about thumbnails etc: https://www.w3schools.com/howto/howto_css_thumbnail.asp

Properties:

  • Image URL
  • Image Width
  • Image Height
  • Background Color
  • Padding
  • Personal Info(link for the thumbnail)

    • JS code that set the above properties in the framework: 

            //default values
        params.imgUrl = params.imgUrl || "pics/redranger.jpg";
        params.imgWidth = params.imgWidth || 100;
        params.imgHeight = params.imgHeight || 200;
        params.backgroundColor = params.backgroundColor || "red";
        params.padding = params.padding || "10px";
        params.personInfo = params.personInfo || "http://powerrangers.wikia.com/wiki/Jason_Lee_Scott";

    How to place the tag element and make it a thumbnail ( example in html): 

                    <a target="_blank" href="http://powerrangers.wikia.com/wiki/Kimberly_Ann_Hart">
                 <img src="pics/pinkranger.jpg" alt="Pink Ranger">
                </a>

    In the framework, you should create a string builder that is the same as the element tags for the thumbnail 

                     function display(){
            var str ="<a target= '_blank' href='"+personInfo+"'/>";
          
            str+="<img src='" + imgUrl + "' width:'" + imgWidth + ";' height: '" + imgHeight +";'/>";
            
            +"</a>";
            
            prObj.innerHTML = str;
            
        }
